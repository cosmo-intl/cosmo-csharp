# Cosmo (.NET) — API Reference

**Cosmo** is an ergonomic, ICU-backed localisation library for .NET.
Install via NuGet: `dotnet add package Miloun.Cosmo`

Browse the API:

- [Cosmo](../api/Miloun.Cosmo.Cosmo.html) — the main class
- [Modifiers](../api/Miloun.Cosmo.Modifiers.html) — calendar / currency / time-zone overrides
- [NumberOptions](../api/Miloun.Cosmo.NumberOptions.html) — number formatting controls
- [CollationOptions](../api/Miloun.Cosmo.CollationOptions.html) — collation tailoring
- [Exceptions](../api/Miloun.Cosmo.CosmoException.html) — error hierarchy
